## Setup
### Requirements:
python <= 3.6

How to Run
===

start the app with:
`source venv/bin/activate`
`python src/main.py`

if you don't have a webcam you can run a sample test with a picture by running:
`python src/test.py`

Run tests with(no tests available yet):
`nosetests test`

Once done, you can shutdown your virtual environment by:
`deactivate`
